#REALEBOHA SEKHITLANE

A modern, responsive portfolio website showcasing my work as a Frontend Developer, Mobile App Developer, and SEO Specialist. Built with React and Tailwind CSS, this portfolio demonstrates clean code, thoughtful UI/UX, and attention to detail.

## 🛠 Technologies Used

- **React.js** - Frontend framework
- **Tailwind CSS** - Styling
- **Vite** - Build tool
- **Lucide React** - Icons
- **Formspree** - Contact form handling

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll navigation
- Mobile hamburger menu
- Scroll to top button
- Animated sections (fade-in, slide-in)
- Project showcase with live demos and GitHub links
- Certifications and achievements section
- Contact form
- CV download
- Dark theme with orange accent (#f39200)
